# NestJS

NestJs is a framework for building modern, scalable, and maintainable service side applications.

It support TypeScript out of the box.

## Features

- It is structured and opininoated.
- It follows MVC design pattern, unlike express which is unstructured and unopinionated.
- It uses Express under the hood so you get express core functionality as well.

### Implementation Details.

1. install node; must have
   - checkout [link](https://nodejs.org/en/download)
2. install nest
   - checkout [link](https://docs.nestjs.com/)
     or follow:
     1. `npm i -g @nestjs/cli`
     2. `nest new project-name` and choose your desire package: `npm`
